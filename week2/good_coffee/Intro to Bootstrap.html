Div and Span

Divs and spans are HTML tags commonly used with CSS.

  • Divs are "block" elements and are used for large chunks of code
  • Spans are "inline" elements and are used for small bits of code  
<div id="content">
    <h1>My Awesome Site</h1>
    <p>You'll find some really cool stuff here.</p>
</div>
#content {
  width: 600px;
  background: #fff;
}

Box Model

Each HTML element can be considered a box. The box model is essentially, you guessed it, a box that wraps around HTML elements!

 

It consists of: margins, borders, padding, and the actual content.

What is Bootstrap?

Bootstrap is a free front-end framework for faster, easier web development. It includes HTML and CSS based design templates as well as JavaScript plugins. 

 

Included features

  • HTML elements
  • CSS classes
  • JavaScript Plugins
  • Responsive designs

Bootstrap

Where does Bootstrap come into play?

Designing UI and producing the necessary custom CSS and HTML can be time consuming. Bootstrap's plug-and-play components and CSS styles allow developers to rapidly prototype applications without spending too much time on front-end concerns. 

Think of it like Legos, but instead of plastic we are using HTML, CSS, and JS code.

Bootstrap

How to include Bootstrap on your site.

Method 1: Download the latest version of bootstrap from getbootstrap.com. Once downloaded you need to use npm to install 

 

Method 2: Use the Bootstrap CDNs to load the Bootstrap classes and necessary JavaScript as outlined in the Bootstrap Quickstart Guide

There are pros and cons to each approach, but we'll be using the second method in this course.

Bootstrap

Bootstrap works almost entirely off of classes. We use premade CSS classes and they automatically style our pages to look fantastic!

 

We can browse the prebuilt classes and components available at: getbootstrap.com. 

For most components in Bootstrap, we only need to copy and paste the code into our webpage!

How does it work?

Bootstrap

Bootstrap gets its responsive design by implementing a grid-based layout system. 

  • The grid consists of 12 columns, evenly spaced
  • By defining how many columns you want each of your elements to take up (using Bootstrap classes), you can create a responsive layout.

Navbar

Here's a few features that you can change in your Bootstrap navbar!

  • Brand (usually a link to home/index page)
  • Dropdowns*
  • Active Class (denotes which page you're currently on)
  • Inline Form (we would need RoR or JS to power)
  • Placeholder (text within input fields)
  • Remove anything that's unwanted

* if Dropdowns are not working, check to make sure jQuery and Bootstrap JS CDN links are written properly
